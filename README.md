# supercollider

Supercollider projects and experiments.

Notes
* Some patches may require [SuperDirt](https://github.com/musikinformatik/SuperDirt). It may be easiest to follow the TidalCycles installation instructions.
* Some patches may require sc3-plugins.
* Under SuperCollider > Preferences > Interpreter, include the absolute path to the `lib` folder in the root of this repo.
